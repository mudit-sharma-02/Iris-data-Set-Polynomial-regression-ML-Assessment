
## 🌸 Iris Dataset: Polynomial Regression Analysis

This repository contains a comprehensive Machine Learning assessment focused on implementing and evaluating **Polynomial Regression** models using the classic Iris Dataset.

### 🚀 Overview

The goal of this project is to predict target variables within the Iris dataset by capturing non-linear relationships that standard linear regression might miss. The notebook covers the end-to-end ML pipeline, from data exploration to model performance metrics.

### 🛠️ Key Features

* **Data Preprocessing:** Handling feature selection and data cleaning for the Iris dataset.
* **Exploratory Data Analysis (EDA):** Visualizing feature correlations and distributions.
* **Polynomial Feature Transformation:** Implementing $n^{th}$ degree polynomial features to improve model fit.
* **Model Training:** Utilizing `Scikit-Learn` for training and hyperparameter tuning.
* **Evaluation Metrics:** Comprehensive analysis using **Mean Squared Error (MSE)** and **R-squared ($R^2$)** scores.

### 📊 Technical Stack

* **Language:** Python
* **Libraries:** `NumPy`, `Pandas`, `Matplotlib`, `Seaborn`, `Scikit-Learn`

### 📈 Results summary

The model demonstrates how increasing the polynomial degree can better capture the variance in petal and sepal measurements, highlighting the balance between **underfitting** and **overfitting** (the Bias-Variance tradeoff).
